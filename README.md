# Mac Studio Setup

### Core Runtime
- **Bun**: Installed via `curl -fsSL https://bun.sh/install | bash`
- **Ruby**: System Ruby + `rbenv` (compiled from source)

### Binaries (~/.bun/bin/)
- **Tailwind CSS v4**: Standalone macOS-arm64 binary
- **LSD**: Rust binary (via cargo or direct download)
- **Starship**: Rust binary (via `curl -sS https://starship.rs/install.sh`)

### Editor
- **Neovim**: macOS arm64 release (appdmg or binary)
